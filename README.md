# Jay Fallon — Portfolio

Personal portfolio site built with Next.js 16, React 19, and TypeScript. Deployed to Railway with Redis for data caching.

## Stack

- **Framework:** Next.js 16 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Data:** Redis on Railway
- **Images:** AWS S3
- **Deployment:** Railway

## Environment Variables

```
REDIS_PUBLIC_URL=redis://...
```

## Development

```bash
npm install
npm run dev
```

## Testing

```bash
npm test
```
